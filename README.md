# BankSystemWebApp

BankSystemWebApp is a web-based banking system built using **HTML/CSS frontend** and **Java backend (Servlets)**.  
It allows users to create accounts, login securely, deposit and withdraw funds, and check balances.

The frontend features a **modern UI**, complete with a **bank logo**, stylish buttons, input forms, and a background image.  
The backend uses Java Servlets to handle account operations and session management.

---

## Features
- Create multiple bank accounts
- Secure login using account number and password
- Deposit money into account
- Withdraw money from account
- Display current account balance
- Attractive and responsive frontend

---

## Technologies Used
- **Frontend:** HTML5, CSS3  
- **Backend:** Java, Servlets, JSP (optional)  
- **Server:** Apache Tomcat  
- **Development Environment:** Eclipse IDE / IntelliJ IDEA
